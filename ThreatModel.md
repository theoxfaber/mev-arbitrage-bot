# MEV Threat Model & Operational Blind Spots

## 1. The Private Mempool / Order Flow Blind Spot

In 2024-2026, 40-50% of DEX volume flows through **private order flow** channels:

- **Wallet RPCs**: MetaMask Swaps, Flashbots Protect, MEV Blocker
- **Intent Protocols**: UniswapX, CoW Protocol, 1inch Fusion
- **Exclusive Builder Deals**: Direct order flow purchases

### Impact
Our `MempoolScanner` relies on `eth_subscribe("newPendingTransactions")` — entirely blind to private order flow. If a massive arb is generated by a UniswapX intent, we won't see it until it's sealed in a block.

### Mitigation
1. **MEV-Share integration** (implemented): Subscribe to order flow auctions for partial-visibility backruns.
2. **On-chain monitoring**: Detect arbitrage after state changes via `eth_subscribe("logs")` on pool contracts.
3. **Direct builder integration**: Negotiate order flow access with wallets/dApps.

## 2. Builder Trust Assumptions

Multi-relay submission (Titan, rsync, Beaverbuild, Flashbots) improves inclusion but requires trusting builders not to unbundle and front-run your MEV. Mitigated by using top-tier, reputable builders only.

## 3. Reorg Risk

The `FlashbotsRelayer` checks canonical confirmation status (>2 confirmations) before aborting backrun attempts. Post-merge, reorgs are rare (< 0.01% of blocks) but still possible during validator set transitions.

## 4. Oracle-Free Slippage Detection

The `SwapDecoder` uses decimal-normalized amounts rather than oracle prices for slippage estimation. This is a deliberate trade-off: oracle queries add latency, and for the purpose of **detecting loose tolerance** (not measuring exact slippage), decimal normalization is sufficient.

## 5. Gas Estimation Risk

Binary-search simulation uses constant-product math which may diverge from actual V3 concentrated-liquidity behavior during large tick crossings. The 20% gas buffer partially mitigates this, but extreme cases could still cause on-chain reverts (which are safe — the flash loan atomically reverts the entire bundle).
